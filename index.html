DAVx⁵ Manual – DRAFT¶

Warning

This is a working draft (source). See the live manual.

This is the manual for DAVx⁵, an CalDAV/CardDAV client and sync app for Android.

The source code of this manual is available. Please send pull requests to make it better!

See also

For more information about DAVx⁵, see the DAVx⁵ Web site. Especially have a look at the tested services and the respective subpages, the Frequently Asked Questions (FAQ), and our forums.
